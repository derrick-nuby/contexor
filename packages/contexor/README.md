# Contexor

Contexor is a lightweight, plug-and-play AI assistant widget that can be embedded into any website—regardless of the tech stack—to provide users with instant, context-aware answers.

## Features

- **No backend required:** Works entirely in the browser.
- **Context-aware:** Automatically gathers relevant content from the site (visible text, metadata, or documents).
- **Easy integration:** Add via script tag or npm package.
- **Customizable:** Use your own AI API key (OpenAI, Claude, etc.) and system prompt.
- **Site-specific answers:** Delivers accurate, context-driven responses.
- **Universal:** Works with any website or web app.
